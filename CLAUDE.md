# brewwater API Docs — Claude Instructions

## Git Workflow

**Immer nach abgeschlossenen Änderungen committen und pushen — ohne zu fragen.**

- Änderungen fertig? → `git add` → `git commit` → `git push` — automatisch, kein "soll ich pushen?"
- Commit-Nachrichten auf Englisch, knapp
- Vercel deployt automatisch nach jedem Push auf `main` (~30s auf developer.brewwater.de)
